# 냉털 / Fridge Raider - Blueprint

## 1. 개요 (Overview)
사용자가 보유한 재료와 선호하는 요리 분야를 입력하면 최적의 레시피를 추천해주는 다국어 웹 애플리케이션입니다. 
- **한국어 서비스 명:** 냉털 (Naeng-Teol)
- **영어 서비스 명:** Fridge Raider

## 2. 프로젝트 상세 (Project Outline)

### 주요 기능 (Features)
- **다국어 지원 (Multi-language Support):** 한국어와 영어 두 가지 버전을 제공하며, 사용자가 실시간으로 전환할 수 있습니다.
- **분야 선택:** 한식, 일식, 중식, 양식 중 택일.
- **수익화 (Google AdSense):** 상단 및 하단 광고 단위 배치 및 `ads.txt` 연동.
- **상세 계량 및 가이드:** 조리 단계별 구체적인 계량(큰술, g 등)과 행동 강령 중심의 상세 지침을 제공합니다.
- **기본 재료 자동 설정 및 편집:** 분야별 필수 양념 자동 추가 및 사용자 삭제 기능.
- **필터링 검색:** 입력된 재료 기반의 최적 매칭 레시피 추천.

### 스타일 및 디자인 (Style & Design)
- **테마:** '프레시 가든(Fresh Garden)' 기반의 현대적인 UI.
- **언어 전환기:** 상단 네비게이션에 배치된 직관적인 언어 토글.

## 3. 현재 구현 계획 (Current Plan)

### 1단계: 다국어 시스템 구축
- `translations.js`: UI 및 데이터 번역본 관리.
- `main.js`: 언어 전환 상태 관리 및 DOM 업데이트 로직.

### 2단계: UI 및 콘텐츠 고도화
- `index.html`, `about.html`, `privacy.html`에 언어 전환 UI 추가 및 텍스트 치환.
- 레시피 데이터의 영어 번역본 보강.

### 3단계: 배포 및 최적화
- GitHub 재배포 및 AdSense 승인 상태 모니터링.
